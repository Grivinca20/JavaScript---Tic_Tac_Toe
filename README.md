A simple project that blends essential concepts in JS, HTML, and CSS, such as:

-Advanced DOM Manipulation: Utilizing querySelectorAll and custom attributes to create a seamless link between the HTML interface and the script;

-Validation Algorithms and Logic: Implementing a verification system based on a matrix of winning combinations;

-Responsive Design with CSS Grid: Crafting a clean and structured layout;

-Event-Driven Programming: Using addEventListener to trigger specific functions for player moves, switching turns, and resetting the game.
